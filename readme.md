# New Dev Machine Set-up Script

## Running the Script
1. Install XCode (includes Command Line Tools for OSX) from App Store
2. Clone this repo: `git clone repo_url ~/.dotfiles`
3. Run `setup.sh` from within `~/.dotfiles`
